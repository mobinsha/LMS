# LMS

## Overview

A Learning Management System (LMS) built with modern web technologies to provide an interactive platform for online education and course management.

### Tech Stack

- **Frontend**: HTML, CSS, JS (for MVP)
- **Backend**: Nest.js,
- **Database**: PostgerSQL, Prisma
- **Authentication**: JWT
